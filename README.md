# Corporate Financial Performance Analysis

## Overview
This project analyses corporate financial performance using Excel and Power BI dashboards. The project focuses on sales trends, profit margins, customer segmentation, and product performance to generate meaningful business insights.

## Tools Used
- Microsoft Excel
- Power BI
- Data Visualization
- Pivot Tables

## Key Features
- Financial dashboard creation
- Sales and profit analysis
- KPI tracking
- Product performance evaluation
- Regional market analysis
- Discount impact analysis

## Outcome
The project demonstrates how data analytics and visualization techniques can support better business decision-making and financial performance monitoring.
